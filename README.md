# نظام دعم القرار الطبي للأورام (MDSS)

نظام متقدم لدعم القرار السريري مبني على أحدث الإرشادات العالمية لعلاج الأورام، مصمم لمساعدة الأطباء في اتخاذ قرارات دقيقة وسريعة.

## المميزات الرئيسية
*   **دعم القرار السريري**: حساب مراحل الورم (TNM Staging) بناءً على المدخلات.
*   **توصيات العلاج**: تقديم توصيات علاجية (Neoadjuvant/Adjuvant) وفقاً للبروتوكولات الدولية.
*   **دعم العمل بدون إنترنت (PWA)**: يمكن تثبيت التطبيق على الجوال أو الكمبيوتر والوصول إليه في أي وقت دون الحاجة لإنترنت.
*   **واجهة مستخدم احترافية**: تصميم عصري يدعم اللغة العربية والإنجليزية مع خاصية الوضع الليلي.
*   **خصوصية البيانات**: يتم تخزين بيانات الحالات محلياً لضمان السرية التامة.

## التقنيات المستخدمة
*   **React + TypeScript**
*   **Vite** (للبناء السريع)
*   **Tailwind CSS + shadcn/ui** (للتصميم)
*   **Vite PWA Plugin** (لدعم العمل أوفلاين)
